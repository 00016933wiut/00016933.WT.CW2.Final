This application is the Recipe Sharing service which allows everyone to add recipes on platform and see them. Attempt was made to implement the deletion and editing methods and basic authorixation methods was in mind to implement. 

To run loccaly:
1) Open Visual Studio Code for the best expirience and pull the repository
2) Open terminal and write "npm init -y"
3) Install dependecies " npm i express pug body-parser method-override"
4) Finally to run, write in terminal "node app.js"


Express: Framework for building web applications with Node.js
Pug: Pug is a templating engine for Node.js and Express.js applications
body-parser: This middleware parses incoming request bodies in a middleware before your handlers, available under the req.body property
method-override: This middleware allows to use HTTP verbs such as PUT or DELETE in places where the client doesn't support it, such as forms or hyperlinks
